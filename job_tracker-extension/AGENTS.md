# Job Tracker Extension Constraints

- Capture only the current tab title and URL.
- Require an explicit user click before sending captured data.
- Do not scrape DOM content.
- Do not inspect webpage HTML.
- Do not inject content scripts.
- Do not request broad browser permissions.
- Do not automate browsing.
- Do not read browser history.
- Do not hardcode job portals.
- Do not implement future-phase features unless explicitly requested.
